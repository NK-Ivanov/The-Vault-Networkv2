## Notes

- **Daily XP Caps:** Starting at Agent rank, daily XP caps apply (Agent: 300, Agent Plus: 300, Verified: 400, Verified Plus: 400, Partner: 500, Partner Plus: 500). Overflow XP is stored and applied on subsequent days.
- **Diminishing Returns:** Repeatable tasks (lessons, quizzes) have diminishing returns:
  - 1st completion: 100% of base XP
  - 2nd-3rd completions: 75% of base XP
  - 4th-5th completions: 50% of base XP
  - 6+ completions: 25% of base XP
- **Daily Tasks:** 2 tasks are shown per day, rotating daily based on day of year.
- **Courses:** Most courses include quizzes that must be passed (typically ≥80%) to earn XP.
- **Login Streaks:** Tracked automatically. Streaks reset if a day is missed.
