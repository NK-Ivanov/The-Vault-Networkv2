## Notes

- **Plus Ranks**: "Plus" ranks (Recruit Plus, Apprentice Plus, etc.) are sub-stages that unlock after completing Stage B tasks. They have the same XP threshold as their base rank but include additional tasks.

- **Task Types**:
  - **Tasks**: Action-based activities (e.g., "Copy Referral Link", "Add Demo Client")
  - **Quizzes**: Knowledge tests (e.g., "Vault Basics Quiz")
  - **Courses**: Educational content with optional quizzes (e.g., "Understanding Automations")

- **XP Rewards**: Range from 100 XP (simple tasks) to 2,000 XP (major milestones like first sale)

- **Unlock Actions**: Some tasks unlock the next stage/rank when completed

- **Login Streak**: Tracked across all ranks, with tasks requiring 2-3 consecutive days of login
