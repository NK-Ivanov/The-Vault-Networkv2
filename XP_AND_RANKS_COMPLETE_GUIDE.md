## Daily Tasks

Daily tasks rotate daily based on the day of the year. Each rank has 5 daily tasks, and 2 are shown each day.

### Recruit (50 XP each)
1. **Open Automations Tab Today** - Navigate to the Automations tab and explore available automations. (50 XP)
2. **View Any 1 Automation Card** - Click on any automation card to view its details. (50 XP)
3. **Open the Support Tab** - Visit the Support tab to see how to get help. (50 XP)
4. **Open Your Referral Link Page** - View your referral link page (view page, not copy link). (50 XP)
5. **Open Overview and Check Today's XP** - Check your XP progress for today in the Overview tab. (50 XP)

**Daily Maximum**: 100 XP (2 tasks × 50 XP)

### Recruit Plus (75 XP each)
1. **Open a Bookmark or Add/Remove One** - Bookmark an automation or manage your bookmarks. (75 XP)
2. **Read Any Automation Brief Section Today** - Read through a complete automation brief section. (75 XP)
3. **Open Support and Read a Message Thread** - Open the Support tab and read through a message thread. (75 XP)
4. **Open Overview → Check Your Weekly XP Trend** - Check your weekly XP trend in the Overview tab. (75 XP)
5. **Open Automations and Filter/Search Once** - Use the filter or search feature in the Automations tab. (75 XP)

**Daily Maximum**: 150 XP (2 tasks × 75 XP)

## Other XP Earning Methods

### Login Streaks
- **Daily Login**: 200 XP per day (automatically awarded when you log in)
- **3-Day Streak Bonus**: +50 XP bonus when you reach 3 consecutive login days
- **7-Day Streak Bonus**: +150 XP bonus when you reach 7 consecutive login days

**Maximum Weekly from Logins**: 1,400 XP (7 days × 200 XP) + 200 XP in bonuses = 1,600 XP

### Diminishing Returns
XP rewards are reduced when repeating activities:
- **1st time**: 100% XP
- **2nd-3rd time**: 75% XP
- **4th-5th time**: 50% XP
- **6+ times**: 25% XP

This applies to lessons, quizzes, and other repeatable activities.

## Rank XP Thresholds

### Rank Progression
1. **Recruit** - 0 XP (Starting rank, 25% commission)
2. **Recruit Plus** - 0 XP (Sub-stage, 25% commission)
3. **Apprentice** - 1,000 XP (30% commission)
4. **Apprentice Plus** - 1,000 XP (Sub-stage, 30% commission)
5. **Agent** - 2,500 XP (33% commission)
6. **Agent Plus** - 2,500 XP (Sub-stage, 33% commission)
7. **Verified** - 4,500 XP (36% commission)
8. **Verified Plus** - 4,500 XP (Sub-stage, 36% commission)
9. **Partner** - 7,000 XP (40% commission)
10. **Partner Plus** - 7,000 XP (Sub-stage, 40% commission)
11. **Partner Pro** - 10,000 XP (45% commission) (Maximum rank)

### XP Required Between Ranks
- Recruit → Apprentice: **1,000 XP**
- Apprentice → Agent: **1,500 XP** (2,500 - 1,000)
- Agent → Verified: **2,000 XP** (4,500 - 2,500)
- Verified → Partner: **2,500 XP** (7,000 - 4,500)
- Partner → Partner Pro: **3,000 XP** (10,000 - 7,000)

**Total XP to Reach Partner Pro**: **10,000 XP**

## XP Summary

### Maximum Daily XP Potential (Partner Pro)
- **Daily Tasks**: 600 XP (2 tasks × 300 XP)
- **Daily Login**: 200 XP
- **Total Daily**: ~800 XP per day

### Maximum Weekly XP Potential (Partner Pro)
- **Daily Tasks**: 4,200 XP (7 days × 600 XP)
- **Daily Logins**: 1,400 XP (7 days × 200 XP)
- **Login Streak Bonuses**: 200 XP (50 + 150)
- **Total Weekly**: 5,800 XP

### Total XP from All Rank Tasks
- **Complete All Tasks (Partner Pro)**: 24,850 XP
- **Note**: This is more than needed to reach Partner Pro (10,000 XP), providing flexibility and allowing for diminished returns on repeats.

### Typical Weekly Earnings
A Partner Pro completing daily tasks and logging in every day:
- **Minimum Weekly**: ~1,600 XP (logins + 2 daily tasks)
- **Maximum Weekly**: 5,800 XP (daily tasks + logins + bonuses)
- **Typical Weekly**: ~2,800-3,500 XP (daily tasks + logins)

### Time to Reach Partner Pro
Assuming consistent daily activity:
- **Fastest**: ~3-4 weeks (completing all tasks + daily activities)
- **Typical**: ~5-7 weeks (consistent daily tasks + logins)
- **Relaxed**: ~10-12 weeks (daily tasks only)
